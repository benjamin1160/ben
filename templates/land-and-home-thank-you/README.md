# Land + Home Package · Thank You

Standalone Next.js project containing only the `/land-and-home/thank-you`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/land-and-home-thank-you&project-name=land-and-home-thank-you
