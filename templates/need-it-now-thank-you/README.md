# Urgent / Time-Pressured · Thank You

Standalone Next.js project containing only the `/need-it-now/thank-you`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/need-it-now-thank-you&project-name=need-it-now-thank-you
