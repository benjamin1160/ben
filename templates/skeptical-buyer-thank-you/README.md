# Skeptical Buyer · Thank You

Standalone Next.js project containing only the `/skeptical-buyer/thank-you`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/skeptical-buyer-thank-you&project-name=skeptical-buyer-thank-you
