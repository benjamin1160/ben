# Priced Out of Traditional · Thank You

Standalone Next.js project containing only the `/priced-out/thank-you`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/priced-out-thank-you&project-name=priced-out-thank-you
