# Working-Class Family · VSL

Standalone Next.js project containing only the `/family-home/vsl`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/family-home-vsl&project-name=family-home-vsl
