# Sick of Renting · Thank You

Standalone Next.js project containing only the `/sick-of-renting/thank-you`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/sick-of-renting-thank-you&project-name=sick-of-renting-thank-you
