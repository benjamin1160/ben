# Retiree / Downsizer · Hybrid

Standalone Next.js project containing only the `/retiree-downsize/hybrid`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/retiree-downsize-hybrid&project-name=retiree-downsize-hybrid
