# Working-Class Family · Quiz

Standalone Next.js project containing only the `/family-home/quiz`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/family-home-quiz&project-name=family-home-quiz
