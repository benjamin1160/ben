# Retiree / Downsizer · Booking

Standalone Next.js project containing only the `/retiree-downsize/booking`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/retiree-downsize-booking&project-name=retiree-downsize-booking
