# Land + Home Package · Application

Standalone Next.js project containing only the `/land-and-home/application`
template, ready to deploy.

## Deploy

```bash
npm install
npm run dev
```

Or deploy directly to Vercel:

https://vercel.com/new/clone?repository-url=https://github.com/benjamin1160/ben&root-directory=templates/land-and-home-application&project-name=land-and-home-application
